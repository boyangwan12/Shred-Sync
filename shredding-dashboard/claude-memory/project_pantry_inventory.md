## Day-type-fit guidance (use this for planning)

When user provides day type, suggest from current pantry:

**Rest day (1400 kcal / 153P / 75C / 55F):**

- 200g tilapia + 100g shrimp = 58P / 5F at low calorie cost
- Veggies fill volume

**Push/pull (1800 kcal / 153P / 100C / 88F):**
- Higher fat budget — beef blade (14F/100g) fits cleanly
- 200g beef blade = 40P / 28F / 420 kcal — substantial single ingredient
- Pair with lean protein (tilapia or shrimp) to round out protein

**Legs day (2000 kcal / 153P / 250C / 43F):**
- **CRITICAL: 43g fat is very tight.** Tilapia (1.7F/100g) is the highest-protein-per-fat option in current stock
- Beef blade is BAD for legs day (14F per 100g would blow fat budget at any reasonable portion)
- Stick to tilapia + shrimp for protein, lots of carbs, minimal added fats
